# Pinoy Cook Book
 An open source website providing some recipes for Filipino dishes made using the Laravel framework. Many types of Filipino Dishes such as:
- Main Dishes
- Desserts
- Soups
- Vegetables
- Breads
- Rice 
- Street Foods
- Sweets
